What is the DOM in web development?

The DOM (Document Object Model) is a programming interface that represents the structure of a webpage. It allows developers to access, manipulate, and update the content, structure, and styles of a website dynamically using JavaScript.

How can you select an element in the DOM using JavaScript?

You can select elements in the DOM using methods like: *document.getElementById('id') Selects an element by its ID. *document.querySelector('.class') Selects the first element that matches a CSS selector. *document.querySelectorAll('.class') Selects all elements matching a CSS selector.

How do you add a new element to the DOM?

1.Use document.createElement('tagName') to create the element.

2.Set its properties using JavaScript (e.g., element.textContent = 'Hello').

3.Append it to a parent element using parent.appendChild(element) or parent.append().

What is the difference between innerHTML and textContent?

1.innerHTML: Gets or sets the HTML content of an element, including tags.

2.textContent: Gets or sets only the text inside an element, ignoring any HTML tags
